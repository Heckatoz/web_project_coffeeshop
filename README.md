# Proyecto Coffee Shop

![Vista principal de la cafetería](./images/inside_coffee_shop.png)

## Descripción

Este proyecto es una página web para una cafetería que incluye:

- Funcionalidad: Uso de enlaces para videos de youtube de rcetas, y un ejemplo de funcionamiento de un formulario para reservas y enlaces dentro de la misma pagina.
- Tecnologías usadas: HTML5, CSS3, metodología BEM

## Planes de mejora

- Agrgar enlaces funcionales a otras paginas y acceso del formulario a una base de datos.
